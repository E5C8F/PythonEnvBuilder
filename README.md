# PythonEnvBuilder
基于bat的python环境部署工具。

# 使用方法：
下载[py部署.bat](https://github.com/E5C8F/PythonEnvBuilder/raw/refs/heads/main/py部署.bat)并执行。py将会部署到脚本工作目录。内置多个py、pip下载源，如果脚本执行速度太慢可以编辑更改下载源，或更换网络环境。
